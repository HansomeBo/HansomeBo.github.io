简历模板

  • 阿里年薪 120 万简历


https://mubu.com/doc3pEw0sOjGI8

1、5 年 JAVA 开发经验，JAVA 基础扎实，熟悉 JVM 原理，具有一定的 JVM 调优经验。
2、熟练使用 SpringMVC、SpringBoot、SpringCloud 能快速搭建微服务。深入理解 SpringCloud 常用组件源码，深入研究过 Eureka、Ribbon、Feign、Hystrix、Zuul 核心源码。
3、深入理解常见设计模式，并在实际项目中深入实践。
4、深入理解分布式事务 ByteTCC 核心源码、可靠消息最终一致性、最大努力通知方案，并有过深入实践。
5、深入理解 zookeeper 分布式锁、Redis 分布式锁及 Curator、Redisson 分布式锁框架核心源码，并基于 Curator 实践过 zookeeper 分布式锁。
6、熟悉分布式接口幂等性保障机制，并在实际项目有过实践。
7、熟悉 Redis 的数据备份恢复方案、高可用架构，对 Nginx + Redis + Ehcache 多层缓存架构方案、缓存雪崩解决方案、缓存重建并发冲突解决方案有过深入实践。
8、深入理解 RocketMQ 和 Kafka 消息中间件，深入研究过 Kafka、RocketMQ 源码。
10、熟悉 Netty 网络中间件，研究过 Netty 源码，深入理解 Reactor 网络模型。
11、熟悉 MySQL 的索引原理 B-树和 B+树，熟悉 Mysql 事物可重复读的底层原理 MVCC 机制，根据 explain 执行计划优化 sql 语句。
12、熟练使用 Sharding-JDBC,中间件对数据分库分表，熟悉动态扩容缩容的分库分表方案。
13、深入理解 Elasticsearch 分布式搜索引擎的架构原理，深入研究过 ES 相关度评分 TF/IDF 算法、倒排索引、深分页 deep paging，能优化 ES 的查询性能。
14、熟悉 CAS 的内部原理，深入研究过 volatile 的底层原理，研究过并发包源码。
